:nl:nl:`&InspectSpecification`
===============================

.. nl:namelist:: InspectSpecification

This namelist block is optional, but must be unique.

It is used to specify the information produced in relation with
:nl:val:`&Process out_type='INSPECT'`. It is ignored when no such output is active.

.. nl:argument::  digits_per_data
    :type: integer
    :default: 0

    Output data with digits_per_data decimal digits.

    0 means G12.4 is used as format

.. nl:argument::  select_record
    :type: integer
    :default: 0

    Output only for record number select_record.

    0 means all records.

.. nl:argument::  select_data_column
    :type: integer
    :default: 0
    
    Print data of grid column select_data_column, may be combined with select_data_row.

    0 means no selection.

.. nl:argument::  select_data_row
    :type: integer
    :default: 0

    Print data of grid row select_data_row, may be combined with select_data_column.

    0 means no selection.

.. nl:argument::  print_file_creation_date
    :type: logical
    :default: .FALSE.

    Print creation date of file content.

.. nl:argument::  print_experiment_version
    :type: logical
    :default: .FALSE.

    Print experiment tag.

.. nl:argument::  print_rec_number
    :type: logical
    :default: .TRUE.

    Print record number.

.. nl:argument::  print_rec_format
    :type: logical
    :default: .FALSE.

    Print record format

.. nl:argument::  print_rec_parameter_table
    :type: logical
    :default: .FALSE.

    Print GRIB table version number.

.. nl:argument::  print_rec_parameter_number
    :type: logical
    :default: .FALSE.

    Print GRIB identifier of parameter.

.. nl:argument::  print_rec_name
    :type: logical
    :default: .TRUE.

    Print record alternate name

.. nl:argument::  print_rec_altname
    :type: logical
    :default: .FALSE.

    Print field short name.

.. nl:argument::  print_rec_level
    :type: logical
    :default: .TRUE.

    Print GRIB level values.

.. nl:argument::  print_rec_level_type
    :type: logical
    :default: .FALSE.

    Print GRIB level type.

.. nl:argument::  print_rec_hgrid_type
    :type: logical
    :default: .FALSE.

    Print type of horizontal coordinates 

.. nl:argument::  print_rec_hgrid_size
    :type: logical
    :default: .FALSE.

    Print size of horizontal grid (ni, nj)

.. nl:argument::  print_rec_hgrid_delta
    :type: logical
    :default: .FALSE.

    Print increment of horizontal grid (di, dj)

.. nl:argument::  print_rec_hgrid_start
    :type: logical
    :default: .FALSE.

    Print start coordinates of horizontal grid (swi, swj) (South-West)

.. nl:argument::  print_rec_hgrid_end
    :type: logical
    :default: .FALSE.

    Print end coordinates of horizontal grid (nei, nej) (North-East)

.. nl:argument::  print_rec_initial_date
    :type: logical
    :default: .TRUE.

    Print initial (reference) date of field.

.. nl:argument::  print_rec_time_range_short
    :type: logical
    :default: .TRUE.

    Print time range, short version (withozt time range type).

.. nl:argument::  print_rec_time_range_long
    :type: logical
    :default: .TRUE.

    Print time range, long version.

.. nl:argument::  print_rec_validation_date
    :type: logical
    :default: .FALSE.

    Print record validation date, as YYYYMMDDHHMM

.. nl:argument::  print_rec_ntuple
    :type: logical
    :default: .FALSE.

    Print n-tuple identificator for current record

    (GRIB1: table version number, parameter number; 

    GRIB2: discipline, category, parameter)

.. nl:argument::  print_rec_pds
    :type: logical
    :default: .FALSE.

    Print product definition section.

.. nl:argument::  print_rec_gds
    :type: logical
    :default: .FALSE.

    Print grid description section.

.. nl:argument::  print_rec_vcoord
    :type: logical
    :default: .FALSE.

    Print vertical coordinates parameters.

.. nl:argument::  print_rec_data
    :type: logical
    :default: .FALSE.

    Print all values of data section.

.. nl:argument::  print_rec_minmax
    :type: logical
    :default: .FALSE.

    Print minimum and maximum of data section.

.. nl:argument::  print_rec_mean
    :type: logical
    :default: .FALSE.


    Print mean value of data section.

.. nl:argument::  print_rec_std
    :type: logical
    :default: .FALSE.

    Print standard deviation of data section.

.. nl:argument::  print_rec_median
    :type: logical
    :default: .FALSE.

    Print median value of data section.

.. nl:argument::  print_rec_minmax_location
    :type: logical
    :default: .FALSE.

    Print location of minimum and maximum in data grid.

.. nl:argument::  print_rec_undef
    :type: logical
    :default: .FALSE.

    Print value of undef as used in data section, and number of undefined values.

.. nl:argument::  print_rec_moment

    :format: comma separated list of integers

    For each value p in the list, print the p-th cental moment of the data section.

.. nl:argument::  print_rec_percentile 

    :format: comma separated list of reals

    For each value p in the list, print the p-th percentile of the data section.

More information on INSPECT pseudo-output is available in section 7.2.
