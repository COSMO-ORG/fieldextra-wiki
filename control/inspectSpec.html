&InspectSpecification#

This namelist block is optional, but must be unique.

It is used to specify the information produced in relation with &Process out_type='INSPECT'. It is ignored when no such output is active.

&InspectSpecification digits_per_data=0 (integer)#

Output data with digits_per_data decimal digits.

0 means G12.4 is used as format

&InspectSpecification select_record=0 (integer)#

Output only for record number select_record.

0 means all records.

&InspectSpecification select_data_column=0 (integer)#

Print data of grid column select_data_column, may be combined with select_data_row.

0 means no selection.

&InspectSpecification select_data_row=0 (integer)#

Print data of grid row select_data_row, may be combined with select_data_column.

0 means no selection.

&InspectSpecification print_file_creation_date=.FALSE. (logical)#

Print creation date of file content.

&InspectSpecification print_experiment_version=.FALSE. (logical)#

Print experiment tag.

&InspectSpecification print_rec_number=.TRUE. (logical)#

Print record number.

&InspectSpecification print_rec_format=.FALSE. (logical)#

Print record format

&InspectSpecification print_rec_parameter_table=.FALSE. (logical)#

Print GRIB table version number.

&InspectSpecification print_rec_parameter_number=.FALSE. (logical)#

Print GRIB identifier of parameter.

&InspectSpecification print_rec_name=.TRUE. (logical)#

Print record alternate name

&InspectSpecification print_rec_altname=.FALSE. (logical)#

Print field short name.

&InspectSpecification print_rec_level=.TRUE. (logical)#

Print GRIB level values.

&InspectSpecification print_rec_level_type=.FALSE. (logical)#

Print GRIB level type.

&InspectSpecification print_rec_hgrid_type=.FALSE. (logical)#

Print type of horizontal coordinates

&InspectSpecification print_rec_hgrid_size=.FALSE. (logical)#

Print size of horizontal grid (ni, nj)

&InspectSpecification print_rec_hgrid_delta=.FALSE. (logical)#

Print increment of horizontal grid (di, dj)

&InspectSpecification print_rec_hgrid_start=.FALSE. (logical)#

Print start coordinates of horizontal grid (swi, swj) (South-West)

&InspectSpecification print_rec_hgrid_end=.FALSE. (logical)#

Print end coordinates of horizontal grid (nei, nej) (North-East)

&InspectSpecification print_rec_initial_date=.TRUE. (logical)#

Print initial (reference) date of field.

&InspectSpecification print_rec_time_range_short=.TRUE. (logical)#

Print time range, short version (withozt time range type).

&InspectSpecification print_rec_time_range_long=.TRUE. (logical)#

Print time range, long version.

&InspectSpecification print_rec_validation_date=.FALSE. (logical)#

Print record validation date, as YYYYMMDDHHMM

&InspectSpecification print_rec_ntuple=.FALSE. (logical)#

Print n-tuple identificator for current record

(GRIB1: table version number, parameter number;

GRIB2: discipline, category, parameter)

&InspectSpecification print_rec_pds=.FALSE. (logical)#

Print product definition section.

&InspectSpecification print_rec_gds=.FALSE. (logical)#

Print grid description section.

&InspectSpecification print_rec_vcoord=.FALSE. (logical)#

Print vertical coordinates parameters.

&InspectSpecification print_rec_data=.FALSE. (logical)#

Print all values of data section.

&InspectSpecification print_rec_minmax=.FALSE. (logical)#

Print minimum and maximum of data section.

&InspectSpecification print_rec_mean=.FALSE. (logical)#

Print mean value of data section.

&InspectSpecification print_rec_std=.FALSE. (logical)#

Print standard deviation of data section.

&InspectSpecification print_rec_median=.FALSE. (logical)#

Print median value of data section.

&InspectSpecification print_rec_minmax_location=.FALSE. (logical)#

Print location of minimum and maximum in data grid.

&InspectSpecification print_rec_undef=.FALSE. (logical)#

Print value of undef as used in data section, and number of undefined values.

&InspectSpecification print_rec_moment#
Format:

comma separated list of integers

For each value p in the list, print the p-th cental moment of the data section.

&InspectSpecification print_rec_percentile#
Format:

comma separated list of reals

For each value p in the list, print the p-th percentile of the data section.

More information on INSPECT pseudo-output is available in section 7.2.
